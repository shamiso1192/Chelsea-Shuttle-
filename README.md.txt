# Chelsea Shuttles Website

This is a responsive website for Chelsea Shuttles, a passenger and goods transport service between Cape Town and Zimbabwe.

## Features
- SEO optimized (meta tags, sitemap, robots.txt)
- HTML5 structure and accessibility
- Client-side form validation with JavaScript
- AJAX form submission simulation
- Interactive gallery with lightbox
- Responsive design with clean UI

## Setup
1. Clone this repository  
   `git clone https://github.com/yourusername/chelsea-shuttles.git`
2. Open `index.html` in your browser.
